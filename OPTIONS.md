#### Format Strings

- Cardano → ADA
``` vim
	#{ada_aud}
	#{ada_cad}
	#{ada_chf}
	#{ada_dkk}
	#{ada_eur}
	#{ada_gbp}
	#{ada_nok}
	#{ada_sek}
	#{ada_usd}
```
